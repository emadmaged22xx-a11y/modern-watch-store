# متجر الساعات الحديثة - Modern Watch Store

> **مشروع جامعي | Full Stack Application | React + Express | السوق السوري**

---

## 1. مقدمة عن المشروع

متجر الساعات الحديثة هو **مشروع جامعي تعليمي** يهدف إلى توضيح مفهوم تطبيق الويب المتكامل (Full Stack) باستخدام:

- **React.js** في الواجهة الأمامية (Frontend)
- **Node.js + Express.js** في الواجهة الخلفية (Backend)

يهدف المشروع إلى بناء متجر ساعات بسيط وعصري يعرض تشكيلة متنوعة من الساعات العالمية في **السوق السوري**، مع إمكانية التصفح والبحث والتفاصيل، وإرسال رسائل تواصل، ولوحة إدارة بسيطة.

## 2. أهداف المشروع

- فهم كيفية التواصل بين **Frontend** و **Backend** عبر **REST API**
- تطبيق التنقل بين الصفحات في **SPA** باستخدام React Router
- استخدام **Axios** لإرسال الطلبات HTTP
- تصميم واجهة مستخدم عصرية باستخدام **Tailwind CSS**
- تخزين البيانات بشكل محلي باستخدام ملفات **JSON**
- دعم اللغة العربية واتجاه **RTL** بشكل كامل
- تنظيم المشروع بشكل احترافي يسهل الشرح الأكاديمي

## 5. شرح Frontend

### المكونات الرئيسية

- **Navbar** - شريط تنقل ثابت يحتوي على روابط كل الصفحات
- **Footer** - تذييل الصفحة يحتوي على معلومات المشروع وروابط التواصل
- **App.tsx** - نقطة دخول التطبيق وتحديد مسارات React Router

### الصفحات

1. **الصفحة الرئيسية (Home)** - Hero Section، تصنيفات المتجر، أحدث الساعات، أكثر المبيعات، مزايا المتجر
2. **صفحة المنتجات (Products)** - عرض البطاقات مع البحث والتصفية والترتيب
3. **صفحة تفاصيل المنتج (ProductDetails)** - صورة كبيرة ومعلومات مفصلة
4. **صفحة من نحن (About)** - تعريف بالمتجر، الرؤية، الرسالة، الأهداف
5. **صفحة التواصل (Contact)** - نموذج إرسال رسائل
6. **لوحة الإدارة (Admin)** - إحصائيات (منتجات، تصنيفات، رسائل) وإدارة المنتجات والرسائل

### الخدمات (Services)

- **api.ts** - جميع دوال الاتصال بالـ API (GET, POST, DELETE)

## 6. شرح Backend

### المسارات (Routes)

- `GET /api/products` - جلب جميع المنتجات
- `GET /api/products/:id` - جلب منتج واحد
- `POST /api/products` - إضافة منتج جديد
- `DELETE /api/products/:id` - حذف منتج
- `GET /api/contact` - جلب جميع الرسائل
- `POST /api/contact` - إرسال رسالة جديدة

### Controllers

- **productController.js** - قراءة وكتابة بيانات المنتجات في JSON
- **contactController.js** - قراءة وكتابة رسائل التواصل في JSON

### التخزين

- **products.json** - يحتوي على 31 منتج تجريبي
- **messages.json** - يحتوي على الرسائل المرسلة من النموذج

## 7. شرح جميع الصفحات

### الصفحة الرئيسية

- قسم Hero مع صورة رئيسية ونص ترحيبي
- قسم تصنيفات المتجر (6 تصنيفات بأيقونات)
- عرض أحدث 4 ساعات
- عرض أكثر 4 ساعات مبيعاً
- قسم مزايا المتجر (جودة، توصيل، دعم، ضمان)
- زر CTA للانتقال للمنتجات

### صفحة المنتجات

- شبكة بطاقات تعرض جميع الساعات
- شريط بحث بالاسم
- قائمة منسدلة للتصفية حسب التصنيف (7 تصنيفات)
- قائمة منسدلة للترتيب حسب السعر أو الاسم
- كل بطاقة تحتوي على صورة، اسم، سعر، تصنيف، توفر، زر تفاصيل

### صفحة تفاصيل المنتج

- صورة كبيرة للساعة
- الاسم والوصف والسعر
- بلد المنشأ والتصنيف وحالة التوفر
- زر العودة للمنتجات

### صفحة من نحن

- تعريف بالمتجر
- الرؤية والرسالة
- أهداف المشروع التعليمية

### صفحة التواصل

- معلومات التواصل (هاتف، بريد، عنوان)
- نموذج باسم + بريد + رسالة
- عرض رسالة نجاح بعد الإرسال
- حفظ الرسالة في Backend

### لوحة الإدارة

- بطاقات إحصائيات (عدد المنتجات + عدد التصنيفات + عدد الرسائل)
- جدول المنتجات مع زر حذف
- جدول الرسائل المستلمة مع التاريخ

## 15. كيفية إضافة منتج جديد

### الطريقة 1: عبر ملف JSON

1. افتح `backend/data/products.json`
2. أضف كائن جديد يتبع نفس البنية:

```json
{
  "id": 32,
  "name": "Apple Watch Ultra",
  "price": 4500000,
  "category": "ساعات ذكية",
  "description": "وصف الساعة بالعربية",
  "image": "https://images.pexels.com/...",
  "country": "الولايات المتحدة",
  "inStock": true
}
```

### الطريقة 2: عبر API

استخدم Postman أو curl:

```bash
curl -X POST http://localhost:5000/api/products \
  -H "Content-Type: application/json" \
  -d '{"name":"Apple Watch Ultra","price":4500000,"category":"ساعات ذكية","description":"وصف الساعة","image":"https://...","country":"الولايات المتحدة","inStock":true}'
```

## 16. كيفية حذف منتج

### عبر لوحة الإدارة

1. افتح `http://localhost:5173/admin`
2. اضغط على أيقونة الحذف (Trash) في جدول المنتجات

### عبر API

```bash
curl -X DELETE http://localhost:5000/api/products/1
```

حيث `1` هو المعرف (id) للمنتج.

تم إنشاء هذا المشروع بهدف **تعليمي** لمساعدة الطلاب على فهم:
